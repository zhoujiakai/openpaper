 **为什么没讲** **LSTM？**                               



 西瓜书定位是机器学习**入门教材**，覆盖的是经典 ML 方法的数学基础（线性模型、决策树

 、SVM、贝叶斯、集成学习、聚类等），神经网络只占一章。LSTM 属于深度学习中 RNN

 的具体变体，不是这本书的覆盖面。当时（2016 年出版）如果想系统学

 LSTM，通常推荐配合 Goodfellow

 的《深度学习》或邱锡鹏的《神经网络与深度学习》来看。



 简单说：西瓜书给的是机器学习的"通识教育"，LSTM 属于深度神经网络的专项内容了。
