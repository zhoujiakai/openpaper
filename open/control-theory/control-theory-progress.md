## 已完成内容 ✅

### PID 控制
- [x] P（比例控制）：Kp × e(t)，稳态误差产生原因
- [x] I（积分控制）：累积历史误差，消除稳态误差
- [x] D（微分控制）：看误差变化趋势，提前刹车抑制超调
- [x] PID 三句话总结：P 管现在 / I 管历史 / D 管趋势
- [x] 震荡处理：Kp 过大导致，减小 Kp
- [x] 积分饱和（Integral Windup）：
  - 原因：执行器饱和时 I 持续累积
  - 解法：积分分离 / Anti-windup / 积分限幅

### MPC 模型预测控制
- [x] MPC vs PID 根本区别：主动规划 vs 被动反应
- [x] 三要素之一：预测模型 `x(k+1)=Ax(k)+Bu(k)`
- [x] 三要素之二：滚动优化 + 代价函数（Q 跟踪 / R 平顺）
- [x] 三要素之三：反馈校正（测真实状态，重新优化）
- [x] 完整执行流程理解：①测量状态 → ②设目标 → ③预测 → ④代价函数 → ⑤加约束 → ⑥解 QP → ⑦执行第一步 → ⑧滚动
- [x] 约束处理：MPC 在框框里挑最优，PID 直接算
- [x] MPC vs LQR：LQR 离线算一次固定 K，MPC 在线每步重新算
