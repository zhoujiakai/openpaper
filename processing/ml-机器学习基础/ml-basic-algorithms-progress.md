## 已掌握的关键知识点

### 模块 1：决策树
- 分裂准则：选信息增益最大的特征+阈值，让每堆更纯
- Gini vs 熵：都量乱度，Gini 不用 log 更快，sklearn 默认 Gini
- 过拟合根源：贪心追纯度，树越深越容易记住噪音
- 剪枝：预剪枝（`max_depth`/`min_samples_leaf`）；后剪枝（`ccp_alpha`，越大砍越多）
- CART：只能二叉分裂，贪心，分类用 Gini

### 模块 2：随机森林
- 两重随机：Bootstrap 采样（每棵树数据不同）+ 随机特征（每次分裂只看部分特征）
- 降的是**方差**，不是偏差（Boosting 降偏差）
- OOB：Bootstrap 约 36.8% 样本未被抽到，可用于不额外划分验证集的泛化估计
- n_estimators：越多越好，但收益递减，100~500 够用

### 模块 3：SVM（已覆盖）
- 最大间隔：找离两边最近的点（支持向量）尽可能远的决策边界
- Hard Margin：要求完全可分，现实不用；Soft Margin：允许越线，C 控制惩罚力度
- C 参数：大→间隔窄→过拟合；小→间隔宽→欠拟合
- 核技巧：把数据映射到高维使其线性可分，核函数避免显式计算高维坐标
- gamma（RBF核）：大→影响范围小→边界扭曲→过拟合；小→边界平滑→欠拟合

## 被纠正过的误区

- ❌ 以为随机森林降的是偏差 → ✅ Bagging 降方差，Boosting 降偏差
- ❌ 以为 SVM 自动忽视异常值 → ✅ 取决于 C 参数，小 C 才容忍越线
