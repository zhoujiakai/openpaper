# C 语言 — 个人笔记

> 初次写笔记时间：20260530周六

记住的内容：

- 熟悉了 C 语言的关键字及含义：const、`typedef struct`、`#define`、等。

- 了解了不同类型的字节数：char 是1个字节、int 一般是4个字节、指针类型是 8 字节（64位操作系统）。

- 了解了 C 语言中栈和堆的区别：函数栈（Linux默认8 MB），用户malloc申请的内存在堆。

- 了解了函数指针的定义的写法：`int (*f)(int, int)`。

- 了解了指针的概念和不同场景下的格式，了解了 `*` 在不同场景下的含义。

- 了解了嵌入式应用中经常会用到的位运算在 C 语言中的写法：`& | ^ << >>`。

- 了解了 C 语言中数组类型的特点：给指针赋值会退化，数组变量不可重新赋值。
