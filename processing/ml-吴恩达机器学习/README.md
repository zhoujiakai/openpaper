每20分钟视频观看、消化、笔记 = 1小时。

统计下来，预估时间公式，视频 : 消化+笔记 = 1 : 2。

照这样算，20小时课程需要60小时才能学完，需要4天。



看完一个视频，必须马上消化、笔记，不能看完一堆视频再消化、笔记。前者是最痛苦、最残忍，同时也是最高效的学习方式。屡试不爽。
