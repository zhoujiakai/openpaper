# 吴恩达机器学习

> 课程来源：[B站 — 吴恩达机器学习](https://www.bilibili.com/video/BV16jyuBBEom)（免费、中英文字幕），2022 新版。
>
> 实验代码获取：联系 B站视频对应账号的客服获取（与本仓库无关）。

## 课程概览

> "吴恩达机器学习"实际是一套专项课程（Specialization），分为三个部分，共 142 节，视频总长 20 小时。

| 课程 | 内容 | 节数 |
|------|------|------|
| 第一部分 | 监督学习：线性回归、逻辑回归、梯度下降、特征工程 | 41 |
| 第二部分 | 神经网络、决策树、偏差与方差、模型选择 | 59 |
| 第三部分 | 聚类、异常检测、推荐系统、强化学习 | 42 |

## 学习建议

- 建议投入为视频时长的 3 倍（观看 + 消化 + 笔记 ≈ 60 小时）
- 每小节视频约 10 分钟，每看完一节就写一节笔记，不要堆到后面，否则容易忘掉前面的内容
- 可以先看完所有视频，再统一过实验代码——先建立整体认知，再动手实践，思路更连贯
- Jupyter notebook 实验代码英文看着费劲的话，可以让 AI 先把 markdown 和注释翻译成中文再读
