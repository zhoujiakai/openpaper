【本集概览】  

「自然光」第一集（下），哲学家赵汀阳，与主持人窦文涛，继续沿着哲学脉络，深入讨论“他人不同意”这一“人类问题总结”。  从戳破“电车悖论”是伪命题，到西方哲学关于“他人之心”的疑问；将东西方哲学的分野归结为“存在”与“变易”之争，西方追问“存在”，而中国智慧则讲求“万物皆是过程”。  赵汀阳的“天下体系”，并非复古野心，而是网格式世界，聚焦世界内部化、关系理性和孔子改进；最后，望向历史，正如渔樵史观的“语不休”——古今多少事，都付笑谈中，不为历史赋予结论，话才能一直说下去。  哲学也是如此，对话永远进行，哲学就会永恒生长。
