## 书籍

### 毛泽东选集

- 分类：政治 / 哲学
- 作者：毛泽东
- 作者介绍：毛泽东（1893—1976），字润之，湖南湘潭人，伟大的马克思主义者，无产阶级革命家、战略家和理论家，中国共产党、中国人民解放军和中华人民共和国的主要创立者。
- 出版年份：1991年（第二版）
- 出版社：人民出版社
- 说明：毛泽东思想的重要载体，系统记录了中国革命与建设时期的理论与实践成果。
- 笔记：[书-毛泽东选集/](书-毛泽东选集/)

### 老子注译

- 分类：哲学
- 作者：高亨
- 作者介绍：高亨（1900—1986），字晋生，吉林双阳人，古文字学家、先秦文化史专家。清华国学研究院毕业，师从梁启超、王国维，曾任教于东北大学、山东大学等校。治学以文字训诂为根底，主张"以朴学释老"。《老子注译》逐章先列原文，再附注释与译文，兼重出土文献与版本对勘，是新中国成立后较早的《老子》今注今译著作。高亨《老子注译》，采用王弼本原文。
- 老子介绍：老子（约公元前571年—约公元前471年），姓李名耳，字聃，春秋时期陈国苦县人，道家学派创始人，曾任周朝守藏室之史（相当于国家图书馆馆长），后骑青牛西出函谷关，应关令尹喜之请著《道德经》五千言。
- 出版年份：2010年8月（清华大学出版社）
- 出版社：清华大学出版社（高亨著作丛刊）
- 笔记：[书-老子注译/](书-老子注译/)
