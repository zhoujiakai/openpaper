## 6 — C1_W2_Lab02 多变量线性回归

- [x] 16.2%，20260529周五

> - 实验：多变量线性回归，[实验/第一课/C1_W2_Lab02_Multiple_Variable_Soln.ipynb](实验/第一课)

多个样本，每个样本有多个特征。用numpy实现多变量的代价计算、多变量的梯度下降计算。

多变量的代价计算，跟单变量代价计算的区别就是 `np.dot(X[i], w) + b`：

```python
def compute_cost(X, y, w, b): 
    """
    compute cost
    Args:
      X (ndarray (m,n)): Data, m examples with n features
      y (ndarray (m,)) : target values
      w (ndarray (n,)) : model parameters  
      b (scalar)       : model parameter
      
    Returns:
      cost (scalar): cost
    """
    m = X.shape[0]
    cost = 0.0
    for i in range(m):                                
        f_wb_i = np.dot(X[i], w) + b           #(n,)(n,) = scalar (see np.dot)
        cost = cost + (f_wb_i - y[i])**2       #scalar
    cost = cost / (2 * m)                      #scalar    
    return cost
```

多变量的梯度下降计算，跟单变量的梯度下降计算的区别就是 `(np.dot(X[i], w) + b) - y[i]`：

```python
def compute_gradient(X, y, w, b): 
    """
    Computes the gradient for linear regression 
    Args:
      X (ndarray (m,n)): Data, m examples with n features
      y (ndarray (m,)) : target values
      w (ndarray (n,)) : model parameters  
      b (scalar)       : model parameter
      
    Returns:
      dj_dw (ndarray (n,)): The gradient of the cost w.r.t. the parameters w. 
      dj_db (scalar):       The gradient of the cost w.r.t. the parameter b. 
    """
    m,n = X.shape           #(number of examples, number of features)
    dj_dw = np.zeros((n,))
    dj_db = 0.

    for i in range(m):                             
        err = (np.dot(X[i], w) + b) - y[i]   
        for j in range(n):                         
            dj_dw[j] = dj_dw[j] + err * X[i, j]    
        dj_db = dj_db + err                        
    dj_dw = dj_dw / m                                
    dj_db = dj_db / m                                
        
    return dj_db, dj_dw
```

## 7 — C1_W2_Lab03 特征缩放与学习率

- [x] 18.9%，20260529周五

> - 实验：特征缩放与学习率，[实验/第一课/C1_W2_Lab03_Feature_Scaling_and_Learning_Rate_Soln.ipynb](实验/第一课)

**特征尺度不同怎么处理**

- 现象：房价预测中不同特征之间的数值尺度可能相差很大，有的特征1000～2000，有的特征1～10。

- 问题：代价函数的等高线会变成狭长的椭圆，梯度下降走锯齿状路线。

- 原理：所有参数的学习率如果设置成一样，并且权重参数随机初始化，有的参数学习过程是大跨步，有的是蜗牛步速。

- 解决：所以需要进行特征缩放。

**多个特征怎么把预测结果可视化**

- 现象：房价预测中有多个特征和一个预测值。

- 问题：怎么绘制图表，可视化预测值和实际值的拟合程度？

- 解决：将多个特征拆开，每个图表只绘制某个特征下，预测值和实际值之间的关系。就能够直观查看拟合程度。

## 8 — C1_W2_Lab04 特征工程与多项式回归

- [x] 21.6%，20260529周五

> - 实验：特征工程与多项式回归，[实验/第一课/C1_W2_Lab04_FeatEng_PolyReg_Soln.ipynb](实验/第一课)

如果我们用3阶多项式回归，实际数据比较适合用2阶多项式拟合，梯度下降会给3阶项和1阶项的权重一个很小的值，给2阶项的权重一个很大的值，梯度下降这种自动强调的效果，会让模型中的二阶项更好地发挥作用。

另一个视角是：线性回归中，我们想要的特征最好是跟目标值成线性关系，如果 x 和 y 不是线性关系，我们就可以构造 $x^2$ 这个特征，可能 $x^2$ 和 y 就是线性关系。就能够很好地用线性回归来预测。

即使数据是一条再扭曲的曲线，我们也能够用更多阶的多项式很好地拟合数据。
